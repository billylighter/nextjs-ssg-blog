# Next.js SSG Blog

A static blog built with **Next.js** using Markdown files and **Static Site Generation (SSG)**.

## Features

- Fetch posts from Markdown (`.md`) files.
- Full **SSG** support with `getStaticProps` and `getStaticPaths`.
- Featured images, tags, author info, and reading time.
- Responsive layout with Tailwind CSS.
- Markdown content rendered to HTML using `remark` and `remark-html`.

## Getting Started

1. **Clone the repo:**

```bash
git clone https://github.com/billylighter/nextjs-ssg-blog.git
cd nextjs-ssg-blog
```

## Install dependencies:
npm install
**or**
yarn install

## Set environment variables:
Create a .env.local file:
PUBLIC_NEXT_APP_URL=https://yourdomain.com

## Run the development server:
npm run dev 
**or**
yarn dev

## Build for production:
- npm run build
- npm run start

**or**
- yarn build
- yarn start
